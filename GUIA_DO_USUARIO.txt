    GUIA RÁPIDO - SISTEMA DE MANUTENÇÃO ALS
    Para: Usuário Final (Seu Pai)
═══════════════════════════════════════════════════════════════

 O QUE VOCÊ RECEBEU:

 Uma pasta com estes arquivos:
   ├── SistemaManutencaoALS.exe  ⭐ O programa (clique neste!)
   └── data/                      (sua planilha está aqui)


═══════════════════════════════════════════════════════════════
 COMO USAR - PASSO A PASSO
═══════════════════════════════════════════════════════════════

1️⃣ ABRIR O SISTEMA
   → Dê duplo clique em: SistemaManutencaoALS.exe
   → Aguarde alguns segundos para abrir
   →  Pronto! A tela principal aparecerá


2️⃣ ADICIONAR NOVA MANUTENÇÃO
   → Clique no botão " Novo Registro"
   → Preencha os campos:
      • Placa (obrigatório)
      • Data de Entrada (obrigatório)
      • Demais campos conforme necessário
   → Clique " Salvar"
   →  Registro adicionado!


3️⃣ EDITAR UMA MANUTENÇÃO
   → Opção 1: Dê DUPLO CLIQUE na linha
   → Opção 2: Selecione a linha e clique "️ Editar"
   → Modifique os campos
   → Clique " Salvar"
   →  Registro atualizado!


4️⃣ EXCLUIR UMA MANUTENÇÃO
   → Selecione a linha (clique nela)
   → Clique "️ Excluir"
   → Confirme a exclusão
   →  Registro removido!


5️⃣ BUSCAR VEÍCULOS
   → Use os campos de filtro no topo:
      • Placa: Digite a placa parcial (ex: "RXP")
      • Veículo: Digite o tipo (ex: "CAVALO")
      • Status: Escolha "EM SERVIÇO" ou "FINALIZADO"
   → Clique " Buscar"
   → Para ver todos novamente: " Limpar"


6️⃣ SALVAR OS DADOS
   → IMPORTANTE: Clique " Salvar" sempre que fizer mudanças!
   → O sistema pergunta ao fechar se quer salvar
   →  Um backup é criado automaticamente na pasta 'backup'


7️⃣ GERAR RELATÓRIO
   → Clique " Relatório"
   → Veja as estatísticas:
      • Total de manutenções
      • Veículos em serviço
      • Tempo médio
   → Relatório salvo em: pasta 'output'


8️⃣ EXPORTAR PARA EXCEL
   → Clique " Exportar"
   → Escolha onde salvar
   →  Nova planilha Excel criada!


═══════════════════════════════════════════════════════════════
 FUNCIONALIDADES AUTOMÁTICAS
═══════════════════════════════════════════════════════════════

 O sistema faz SOZINHO:

    DIAS EM MANUTENÇÃO
      → Calcula automaticamente
      → Se ainda está em serviço, conta até hoje

    STATUS
      → "EM SERVIÇO" = Data Entrada preenchida + Data Saída vazia
      → "FINALIZADO" = Ambas as datas preenchidas

    BACKUP AUTOMÁTICO
      → Toda vez que salva, cria cópia de segurança
      → Salvo em: pasta 'backup'


═══════════════════════════════════════════════════════════════
 ENTENDENDO AS CORES
═══════════════════════════════════════════════════════════════

    AMARELO  = Veículo EM SERVIÇO (ainda na oficina)
    VERDE    = Manutenção FINALIZADA


═══════════════════════════════════════════════════════════════
 ESTATÍSTICAS NO TOPO
═══════════════════════════════════════════════════════════════

A barra superior mostra:

    Total        → Total de manutenções registradas
    Em Serviço   → Quantos veículos estão na oficina agora
    Finalizados  → Quantas manutenções foram concluídas
   ⏱️  Tempo Médio  → Média de dias em manutenção
    Placas       → Quantos veículos diferentes


═══════════════════════════════════════════════════════════════
️ DICAS IMPORTANTES
═══════════════════════════════════════════════════════════════

    Sempre clique " Salvar" após fazer mudanças
   
    Duplo clique em uma linha para editar rapidamente
   
    Os backups ficam na pasta 'backup' (segurança!)
   
    Data Entrada é OBRIGATÓRIA
   
    Se não preencher Data Saída, o sistema entende que
      o veículo ainda está em manutenção


═══════════════════════════════════════════════════════════════
 PROBLEMAS COMUNS
═══════════════════════════════════════════════════════════════

 "Programa não abre"
   → Verifique se a pasta 'data' está junto com o .exe
   → Tente executar como Administrador (botão direito)


 "Erro ao salvar"
   → Feche o Excel se estiver com a planilha aberta
   → O sistema não consegue salvar se o arquivo está aberto


 "Não encontra os dados"
   → A planilha deve estar em: data/PROGRAMAÇÃO MANUTENÇÃO (CÓPIA 2).xlsx
   → Não renomeie a pasta 'data'
